# Installing Motivational Screensaver

## Quick Install (recommended)

1. **Download** `MotivationalScreensaver.zip` from the [Releases page](../../releases/latest)
2. **Unzip** the file (double-click or right-click → Open)
3. **Double-click** `MotivationalScreensaver.saver`
4. macOS will ask — click **Install for this user only** (or "Install for all users")
5. Open **System Settings → Screen Saver** and select **MotivationalScreensaver**

That's it! 🎉

---

## Gatekeeper warning (first run only)

Because the screensaver isn't from the Mac App Store, macOS may show a warning like:

> *"MotivationalScreensaver.saver" cannot be opened because it is from an unidentified developer.*

**Fix:**

1. Open **System Settings → Privacy & Security**
2. Scroll down to the Security section
3. You'll see *"MotivationalScreensaver.saver was blocked…"* — click **Allow Anyway**
4. Try double-clicking the `.saver` file again and click **Open** on the next prompt

You only need to do this once.

---

## Uninstall

Open **System Settings → Screen Saver**, right-click **MotivationalScreensaver** in the list, and choose **Delete Screen Saver**. Or delete manually:

```
~/Library/Screen Savers/MotivationalScreensaver.saver
```

---

## Customising

When the screensaver is selected in System Settings, click **Screen Saver Options…** to:

- Toggle quote **categories** (Success, Mindset, Health, Leadership)
- Adjust how often quotes **rotate** (5 – 60 seconds)

---

## Requirements

- macOS 12 Monterey or later
- Internet connection (falls back to built-in quotes if offline)
